# Financial Dashboard

A modern financial dashboard built with Next.js and TypeScript that visualizes transactions using charts and provides basic admin functionality.

---

## Features

- Line chart for financial trends  
- Pie chart for category distribution  
- Transaction list view  
- Viewer role to view transactions  
- Admin role to add transactions  
- Clean and responsive UI  

---

## Landing Page

A separate landing page with smooth animations using:

- GSAP (animations)  
- Lenis (smooth scrolling)  

---

## Tech Stack

- Next.js  
- TypeScript  
- Tailwind CSS  
- Zustand (state management)  
- Recharts (data visualization)  
- GSAP + Lenis (animations)
